Project On Console

Visual Studio is a coding platform created by Microsoft. It has many extensions and you can use it to do a wild range of coding.

To set up visual studio you have to go to the website and choose which type of laptop you are using. Next download or install the platform. Later on, create a folder for your specific topic. Then press "new file", and select your folder. Then start coding!

A Console Screen allows you to input commands and receive visual output from a computer or computer system.
